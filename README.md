✅ Lista de Tarefas (To-Do List)

Este projeto consiste em uma aplicação simples de Lista de Tarefas (To-Do List) desenvolvida utilizando HTML, CSS e JavaScript.

A aplicação permite que o usuário adicione, visualize e gerencie tarefas, ajudando na organização de atividades do dia a dia.

O objetivo do projeto é praticar conceitos fundamentais do desenvolvimento front-end, como manipulação do DOM, eventos em JavaScript e organização de interface com CSS.

🚀 Tecnologias Utilizadas

HTML5 – Estrutura da aplicação

CSS3 – Estilização da interface

JavaScript (Vanilla JS) – Lógica da aplicação e manipulação do DOM

🎯 Funcionalidades

✔️ Adicionar novas tarefas
✔️ Listar tarefas na tela
✔️ Marcar tarefas como concluídas
✔️ Remover tarefas da lista
✔️ Interface simples e intuitiva
